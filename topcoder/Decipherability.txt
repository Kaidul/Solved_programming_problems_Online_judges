Cat Snuke will remove exactly <b>K</b> characters from the string.
Afterwards, Snuke will show you the new string and you have to guess the original indices of all removed characters.
You win the game if you guess all of them correctly.<br></br><br></br>

For example, suppose that <b>s</b>="snuke" and <b>K</b>=2.
Snuke removed two characters and showed you the string "nue".
In this situation you can easily deduce that Snuke must have removed the characters <b>s</b>[0] and <b>s</b>[3].
You announce that the indices of removed characters are 0 and 3, and you win the game.<br></br><br></br>

You are given the string <b>s</b> and the int <b>K</b>.
Return "Certain" (quotes for clarity) if you can always be sure to win the game, regardless of which characters Snuke removes.
Otherwise, return "Uncertain".
